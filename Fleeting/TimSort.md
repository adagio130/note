#algorithm
- Timsort is a hybrid of Mergesort and Insertion sort

Time complexity

| Algorithm | Best       | Average    | Worst      |
| --------- | ---------- | ---------- | ---------- |
| Quicksort | O(n log n) | O(n log n) | O(n^2)     |
| Timsort   | O(n)       | O(n log n) | O(n log n) |


Space complexity

| Quicksort | Timsort |
| --------- | ------- |
| O(log n)  | O(n)    |
